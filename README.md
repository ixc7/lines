
## lines

```

lines 0.0.0
prints a range of lines from a file
if no options are specified, prints the first line of file

USAGE 
  lines [OPTIONS] FILE...

OPTIONS
  -r, --range <a-b>
      print lines within range
  -f, --from <n>
      start at line
  -t, --to <n>
      end at line
  -h, --help
      print this text and exit
  -v, --version
      print version and exit (0.0.0)

```
